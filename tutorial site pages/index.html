Welcome to our tutorial everyone!

Today, we will teach you how to create a web page - using a basic implementation of 3-tier architecture in ASP.NET with C#. Before we begin, we will give you a brief background on three-tier architecture:

Three tier architecture is a type of client-server structure where the user interface, functional process logic ("business rules"), and computer data storage (also data access), are maintained as independant parts. This architecture is also a software architecture and software design pattern. A more in-depth explanation of three-tier architecture can be found by visiting this link.
